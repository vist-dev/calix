# Calix

動画編集ソフト向けの**差分・バージョン管理ライブラリ**。

編集操作のメタ情報（文字列・数値）のみを差分として記録し、git ライクなコミット・ブランチ・マージ・リベースを提供する。動画・音声ファイルなどのバイナリは一切扱わない。

## 特徴

- **サブモジュール単位のブランチ管理** — クリップ・エフェクト・トランジションなど、編集要素ごとに独立したバージョン履歴を持つ
- **差分ベースの状態管理** — 全状態のスナップショットではなく、変更差分（set/remove）のチェーンで履歴を表現する
- **三方向マージとリベース** — 共通祖先からの差分比較によるコンフリクト検出、Fast-Forward 対応
- **アトミック書き込み** — tmp→rename パターンによるクラッシュセーフなファイル永続化
- **FFI 対応ビルド** — `staticlib` / `cdylib` として C++ から呼び出し可能
